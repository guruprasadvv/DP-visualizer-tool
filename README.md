The Dynamic Programming (DP) Visualizer is an interactive tool designed to help users understand dynamic programming algorithms through step-by-step visualization. It demonstrates how solutions are built using recursion, memoization, and tabulation, making complex concepts easier to grasp.

Features:
1)Step-by-step DP table visualization
2)Recursive call tracing
3)Play / Pause / Next execution controls
4)Clear representation of state transitions
5)Multiple algorithm support


Algorithms Implemented:
1)Longest Common Subsequence (LCS)
2)0/1 Knapsack
3)Floyd-Warshall Algorithm
4)Subset Sum
5)Matrix Chain Multiplication
6)Traveling Salesman Problem (TSP)
7)Palindrome Partitioning
