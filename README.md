The Dynamic Programming (DP) Visualizer is an interactive tool designed to help users understand dynamic programming algorithms through step-by-step visualization. It demonstrates how solutions are built using recursion, memoization, and tabulation, making complex concepts easier to grasp.

Features:

Step-by-step DP table visualization  ,  Recursive call tracing  , Play / Pause / Next execution controls  , Clear representation of state transitions  , Multiple algorithm support


Algorithms Implemented:

1)Longest Common Subsequence (LCS)

2)0/1 Knapsack

3)Assembly Line Scheduling

4)Floyd-Warshall Algorithm

5)Subset Sum

6)Matrix Chain Multiplication

7)Traveling Salesman Problem (TSP)

8)Count Subsets with Sum k

9)Palindrome Partitioning
